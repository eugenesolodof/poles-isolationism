# Data Directory

Данная директория содержит все используемые в проекте данные, включая исходные и обработанные файлы для анализа парламентских голосований по развертыванию и выводу вооруженных сил в рамках международных операций.

---

## Источники данных

1. **Parliamentary Deployment Votes Database (PDVD)**
   - **Описание:** База данных содержит информацию о голосованиях парламентских партий по законопроектам, связанным с участием вооруженных сил в международных операциях.
   - **Переменные:** 
     - `support` – доля голосов за законопроект.
     - `gov` – участие партии в правительстве.
     - `wthdr` – характер голосования (развертывание/вывод).
     - `alliance` – членство страны в военном альянсе.
     - `un` – участие в миссиях по резолюциям ООН.
     - `nato` – участие в миссиях НАТО.
     - `eu` – участие в миссиях под эгидой ЕС.
   - **Источник:** [PDVD v3](http://deploymentvotewatch.eu)
   - **Дополнительно:** Полный список миссий доступен [по ссылке](http://deploymentvotewatch.eu/wp-content/uploads/2021/07/PDVD_v3_list-of-missions.xlsx).

2. **Chapel Hill Expert Survey (CHES)**
   - **Описание:** Экспертная оценка партий по идеологическому спектру «левые–правые».
   - **Переменная:** `lrgen` (нормированная шкала 0–10, где 0 = крайне левые, 10 = крайне правые).
   - **Источник:** [Chapel Hill Expert Survey](https://www.chesdata.eu)

3. **Manifesto Project Database (MARPOR)**
   - **Описание:** Контент-анализ партийных манифестов для получения идеологических позиций и электоральной поддержки.
   - **Переменные:** 
     - `rile` – (нормированная шкала 0–10, где 0 = крайне левые, 10 = крайне правые).
     - `pervote` – доля голосов на последних выборах.
   - **Источник:** [Manifesto Project](https://manifestoproject.wzb.eu)

4. **Database of Political Institutions (DPI) 2020**
   - **Описание:** Информация о концентрации власти в правительственных коалициях.
   - **Переменная:** `herf` – индекс Херфиндаля-Хиршмана, выражающий уровень монополизации правительственной коалиции.
   - **Источник:** [DPI 2020](https://data.iadb.org/dataset/the-database-of-political-institutions-dpi-2020)

5. **Источники по международным операциям**
   - **Описание:** Операции классифицированы по типу и характеру миссии на основании открытых источников: официальные отчёты ООН, НАТО, Евросоюза, а также академические публикации о миротворческих и военных операциях.
   - **Переменные:** 
     - `combat`, `cooperative`, `humanitarian` – тип операции.

## Переменные

| Переменная | Описание |
|------------|----------|
| `support` | Доля членов парламентской фракции, поддержавших законопроект, санкционирующий начало или завершение участия национальных ВС в международной операции. Значения варьируются от 0 до 1. |
| `rile / lrgen` | Положение партии на шкале «левые–правые», нормировано от 0 до 10. |
| `gov` | Правительственный статус партии: 1, если партия входила в состав правящей коалиции на момент голосования, 0 — в противном случае. |
| `wthdr` | Характер голосования: 1 для голосований о выводе войск, 0 для развертывания войск. |
| `alliance` | Членство страны в военном альянсе или двустороннем оборонительном соглашении. |
| `herf` | Индекс Херфиндаля-Хиршмана для концентрации власти в правительственной коалиции; от 0 до 1, где большее значение указывает на большую монополизацию власти. |
| `pervote` | Доля голосов, полученная партией на последних парламентских выборах перед голосованием. |
| `postcom` | Историко-политический контекст: 1 для стран бывшего советского блока, 0 — для остальных. |
| `un` | Операция санкционирована резолюцией Совета Безопасности ООН. |
| `nato` | Операция под руководством НАТО или санкционированная Альянсом. |
| `eu` | Операция в рамках Общей политики безопасности и обороны (ОПБО) ЕС. |
| `combat` | Операция с высокой интенсивностью боевых действий и риском потерь. |
| `cooperative` | Операция кооперативного характера (военно-техническое сотрудничество, обучение, логистика). |
| `humanitarian` | Операция с гуманитарными целями, высокой международной легитимностью и низким непосредственным риском. |# Описание переменных

## Географическая выборка

Анализ охватывает 21 демократическую страну: Австралия, Бельгия, Великобритания, Германия, Дания, Ирландия, Испания, Италия, Канада, Литва, Нидерланды, Республика Корея, Румыния, Словакия, США, Турция, Финляндия, Франция, Хорватия, Чешская Республика, Япония.

---

## Примечания

Ввиду различий в формулировках парламентских мандатов между странами, большинство операций были закодированы на агрегированном уровне — согласно официальным целям всей международной кампании. Для части миссий, однако, удалось выделить и закодировать специфические задачи, возложенные на контингент конкретной страны.
